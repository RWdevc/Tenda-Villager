Tenda Villager

Projeto de estudos desenvolvido para praticar HTML, CSS e, futuramente, JavaScript e Python.

A ideia é criar uma loja inspirada no universo de Minecraft, onde jogadores possam adquirir itens, ferramentas, mapas, livros e recursos utilizando uma moeda virtual chamada EMDcoin.

Tecnologias:

- HTML
- CSS
- JavaScript (em desenvolvimento)
- Python (planejado)
- Banco de Dados (planejado)

Funcionalidades:

- Página inicial
- Catálogo de ferramentas
- Catálogo de suprimentos
- Home
- Loja Mística

Em desenvolvimento:

- Sistema de navegação
- Sistema de Login
- Cadastro de usuários
- Sistema de moedas (EMDcoin)
- Carrinho de compras
- Banco de dados
- Painel administrativo
